Escaninho: Resumo

O que é

O recurso Escaninho permite que professores e alunos compartilhem documentos dentro de uma pasta privada para cada aluno. A Escaninho funciona como recursos para permitir que você envie muitos tipos de arquivos e vários arquivos de uma vez. A Escaninho também permite pastas dentro de pastas.

Principais conceitos

Relatórios de andamento: Os instrutores podem usar a ferramenta de Escaninho para fornecer relatórios particulares para cada aluno na classe.

Adicionar participantes: Quando você adiciona os participantes ou alunos em um site, automaticamente ou manualmente, eles terão pastas Escaninho por padrão (se a Escaninho estiver ativada no momento).

Considerações

  • Um novo item da Escaninho irá automaticamente para dentro da pasta que você abrir quando você criar o item.

  • Para documentos de texto simples, páginas HTML e links da web que você cria na interface Escaninho, você pode editar o conteúdo, bem como informações sobre o item, como seu nome ou descrição. Para os arquivos enviados, você pode editar apenas as informações sobre o item, não o conteúdo, para alterar o conteúdo, você deve carregar uma nova versão do arquivo.

  • Sempre que você adicionar um item para a Escaninho, você tem a opção de enviar um email de notificação automaticamente.

  • Certifique-se que conhece as leis e políticas sobre o uso de material com direitos autorais.
